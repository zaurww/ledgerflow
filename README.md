# LedgerFlow

Open-source бухгалтерская система для малого и среднего бизнеса.
Строится с нуля под реальные нужды компаний в СНГ.

**Статус:** Фаза 1 — MVP для сервисной компании 🚧

---

## Почему ещё одна бухгалтерская система?

- 1С — дорогая, закрытая, плохо адаптируется под местное законодательство
- Odoo/ERPNext — строились под западные стандарты, локализация всегда "костыль"
- Excel — не масштабируется, нет аудит-следа, нет двойной записи

LedgerFlow строится с принципом: **законодательство = конфигурация**, а не код.
Новый налоговый стандарт → новое правило в интерфейсе, не новый деплой.

## Быстрый старт

```bash
# Клонировать
git clone https://github.com/your-username/ledgerflow.git
cd ledgerflow

# Запустить (нужен Docker)
docker-compose up -d

# Открыть API документацию
open http://localhost:8000/docs
```
