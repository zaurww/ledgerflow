## [2026-03-29] — Инициализация проекта

### Создано
- Структура папок проекта
- CLAUDE.md — главная инструкция для AI-ассистента
- ARCHITECTURE.md — техническая архитектура
- CHANGELOG.md — этот файл
- README.md — описание для пользователей
- docker-compose.yml — запуск окружения

### Решения принятые сегодня
- **Фаза 1: сервисная компания** — начинаем с простого, без склада и производства
- **Python + FastAPI + PostgreSQL** — стек выбран за читаемость и экосистему
- **Modular monolith** — не микросервисы, один человек не потянет операционную сложность
- **AGPL-3.0** — открытый код, но SaaS-копии тоже должны быть открыты
- **UUID везде** — первичные ключи UUID, не serial (портируемость, нет конфликтов)
- **Decimal для денег** — никогда float, только Decimal

### Следующий шаг
Создать core/ledger — базовые модели и сервис двойной записи.
